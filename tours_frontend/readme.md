# TripNest - Frontend Web App

This is the frontend single page application (SPA) for the TripNest platform, built using React, Vite, Redux Toolkit, and Tailwind CSS.

## Features
- **Modern Dashboard**: Clean card layout showcasing available tour packages.
- **Search & Filters**: Instantly filter tours by location, price range, lodging rating, and transport type.
- **Booking Flow**: Smooth ticket reservation system with Stripe checkout integration.
- **Admin Panel**: Dedicated views for administrators to manage tours, locations, hotels, transport options, and bookings.
- **Clean Styling**: Beautiful design built using Tailwind CSS and Lucide React icons.

## Running the Web App

### Setup Environment
Create a `.env` file in the root of this folder:
```env
VITE_BASE_URL=http://localhost:8080
```

### Install & Start
Run the following commands:
```bash
npm install
npm run dev
```

The web app will run on: `http://localhost:5173`
